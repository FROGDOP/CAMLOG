# CamLog — Rapport caméra

Ouvre `index.html` dans un navigateur.

Chaque projet contient plusieurs journées de tournage. Les cartes et leurs prises sont rattachées à une journée, ce qui permet de les retrouver facilement par date.

## Installation sur iPhone

Une fois l'app publiée sur une adresse HTTPS, ouvre-la dans Safari sur l'iPhone, touche **Partager**, puis **Sur l'écran d'accueil**. CamLog s'ouvrira ensuite comme une application et restera disponible hors ligne.
